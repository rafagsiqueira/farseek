# Contributing to Farseek

Welcome and thank you for wanting to contribute!

## Get started

- Found a bug? [Report it here ➡️](https://github.com/farseek/farseek/issues/new?assignees=&labels=bug%2Cpending-decision&projects=&template=bug_report.yml)
- Have a feature idea? [Submit it here ➡️](https://github.com/farseek/farseek/issues/new?assignees=&labels=enhancement%2Cpending-decision&projects=&template=feature_request.yml)
- Want to add a feature, fix bug, refactor something, or add CI tooling?
  1. Check if there is an [open issue with the `accepted` and `help wanted` labels](https://github.com/farseek/farseek/issues?q=is%3Aopen+is%3Aissue+label%3Aaccepted+label%3A"help+wanted"),
  2. Comment on the issue that you want to work on it,
  3. Wait for a maintainer to assign it to you,
  4. Then [submit your code here ➡️](https://github.com/farseek/farseek/compare)

Check out our [Contributing FAQ](contributing/FAQ.md) and [Development Guide](contributing/DEVELOPING.md) for more detailed information.

## Contributors

Anyone can contribute to Farseek by submitting an issue, helping track down a bug, opening a PR, or help in any way to advance the Farseek project.
